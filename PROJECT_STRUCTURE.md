# The Kingdoms - Official Website

Website resmi untuk server Minecraft komunitas **The Kingdoms**, tempat di mana pemain membina dunia, menciptakan sejarah, dan mengembangkan legasi mereka.

## 📁 Struktur Proyek

```
thekingdoms-main/
├── index.html                    # Halaman utama
├── css/
│   └── styles.css               # Stylesheet utama
├── js/
│   └── script.js                # JavaScript utama
├── images/                      # Folder gambar/aset
│   ├── header-bg.jpeg          # Background header
│   ├── server-bg.jpg           # Background server section
│   ├── survival.png            # Gambar The Kingdoms Survival
│   ├── elpika.png              # Gambar ElpikaCraft
│   ├── season2.png             # Gambar Season 2
│   ├── bedrock.png             # Gambar Bedrock Edition
│   ├── pipit.png               # Profile pipit
│   ├── karipapdap.png          # Profile Karipapdap
│   └── eliana.png              # Profile ElianaRiqma
├── projects/                    # Server projects
│   ├── TheKingdomsJavaSeason1/ # The Kingdoms Season 1
│   ├── ElpikaCraft/            # ElpikaCraft (BCG+)
│   ├── bedrock-players/        # Daftar Pemain Bedrock Edition
│   └── playerss1/              # Player Season 1
├── assets/                      # Additional assets (jika diperlukan)
├── README.md                    # Dokumentasi proyek
└── CNAME                        # GitHub Pages domain config
```

## 🎮 Server yang Tersedia

### 1. **The Kingdoms Survival (Season 1)**
- **Tanggal**: 7 September 2024 – 31 Januari 2025
- **Tipe**: Vanilla Minecraft dengan dukungan komunitas
- **Fitur**: Sistem ekonomi, pertempuran, pembangunan kota

### 2. **ElpikaCraft (BCG+)**
- **Tanggal**: 15 Januari 2025 – 30 April 2025
- **Tipe**: Modpack BigChadGuy Plus
- **Fitur**: Mod-mod berkualitas tinggi, gameplay yang lebih kompleks

### 3. **The Kingdoms Season 2**
- **Tanggal**: 30 April 2025 – 20 Disember 2025
- **Tipe**: 100% Vanilla (Tanpa plugin/mod)
- **Fitur**: Pengalaman pure Minecraft asli

### 4. **The Kingdoms Bedrock Realms**
- **Tanggal**: 30 Juni 2025 – Sekarang
- **Tipe**: Minecraft Realms (Private & Eksklusif)
- **Fitur**: Achievements aktif, tanpa cheats, 100% vanilla

## 📝 Panduan Penggunaan

### Update Konten

1. **Mengubah CSS**: Edit file di folder `css/styles.css`
2. **Menambah JavaScript**: Tambahkan ke `js/script.js`
3. **Menambah Gambar**: Letakkan di folder `images/`
4. **Membuat Halaman Baru**: Buat file HTML baru di root atau folder yang sesuai

### Deploy ke GitHub Pages

```bash
git add .
git commit -m "Update website"
git push origin main
```

Website akan otomatis di-deploy setelah push.
